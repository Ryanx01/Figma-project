# Linktree

## Description
Ce projet est une page web personnelle inspirée de Linktree, permettant de centraliser et de partager plusieurs liens depuis une seule page.

## Installation

Accédez au répertoire du projet :

Utilisation
Ce projet présente une page web avec un design simple et épuré. Voici ce que contient chaque section :

Header : Un message de bienvenue.
Main : Une image ronde et trois liens cliquables.
Footer : Un copyright.

Pour visualiser la page, ouvrez le fichier index.html dans un navigateur. Vous devriez voir une page avec une image ronde et trois liens.
